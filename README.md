# Network View Test

Testprojekt zur Visualisierung eines einfachen Organisationsnetzwerks mit Gephi Lite.

## Modell

Das Netzwerk bildet folgende Struktur ab:

- Personen
- Gruppen
- Mitgliedschaften
- Leitungsfunktion

Beziehungen sind gerichtet:
- `MITGLIED_IN`
- `LEITET`

## Struktur

- Knoten-Typen:
  - `person`
  - `group`
- Kanten:
  - Person → Gruppe

## Zweck

Demonstration einer einfachen, gerichteten Organisationsstruktur mit Mehrfachzugehörigkeiten und Leitungsbeziehungen.
